# Retiro de saldo

## Resuelve la feature de el retiro de saldo

## Que hace

Permite Retirar Saldo para que el usuario se beneficie de sus stocks

## Como funciona

Se añade un servicio en el back para acceder a la base de datos y retirar dinero el cual luego usa el front cada vez que el usuario lo desee y cuando se hace se añade ese dinero a su cuenta bancaria
